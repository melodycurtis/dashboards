Python, R and SQL in Databricks Notebooks

Creating a Notebook is the most intuitive way to work if you are new to Databricks. It provides real-time co-authoring in multiple languages, automatic versioning, and data visualization.

SQL Editor is similar to SQL Server Management Studio (SSMS). It provides a simple interface to run queries on data.
